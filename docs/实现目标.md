## 1. 现有架构分析

### 1.1 编译器管道全景

```
SourceText (字符序列 + 行列映射)
    │
    ▼
Lexer (字符 → Token)
    │  ReadTrivia()  → whitespace, // comments, /* */ comments, line breaks
    │  ReadToken()   → keywords, identifiers, numbers, strings, operators
    │  Diagnostics   → bad character, unterminated strings/comments
    │
    ▼
Parser (Token → 语法树)
    │  ParseCompilationUnit()
    │    ├── ParseMembers()       → MemberSyntax[]
    │    │   ├── ParseFunctionDeclaration()   → FunctionDeclarationSyntax
    │    │   └── ParseGlobalStatement()       → GlobalStatementSyntax
    │    └── ParseEndOfFile()
    │  ParseStatement()  (switch on Current.Kind)
    │    ├── ParseBlockStatement()
    │    ├── ParseVariableDeclaration()   (let/var)
    │    ├── ParseIfStatement()
    │    ├── ParseWhileStatement()
    │    ├── ParseDoWhileStatement()
    │    ├── ParseForStatement()
    │    ├── ParseBreakStatement()
    │    ├── ParseContinueStatement()
    │    ├── ParseReturnStatement()
    │    └── ParseExpressionStatement()
    │  ParseExpression()
    │    └── ParseAssignmentExpression()
    │          └── ParseBinaryExpression(parentPrecedence)  ← precedence climbing
    │                └── ParsePrimaryExpression()
    │                      ├── ParseParenthesizedExpression()
    │                      ├── ParseBooleanLiteral()
    │                      ├── ParseNumberLiteral()
    │                      ├── ParseStringLiteral()
    │                      └── ParseNameOrCallExpression()
    │                            ├── ParseCallExpression()     (if peek '(')
    │                            └── ParseNameExpression()     (otherwise)
    │
    ▼
Binder (语法树 → BoundTree + 符号解析 + 类型检查)
    │  BindGlobalScope()
    │    ├── BindFunctionDeclaration()     → declare symbols
    │    └── BindGlobalStatement()         → bind statements
    │  BindProgram()
    │    └── for each function:
    │          ├── BindStatement(body)     → BoundBlockStatement
    │          ├── Lowerer.Lower()         → desugared IR
    │          └── ControlFlowGraph.AllPathsReturn()  → validation
    │  Expression binding:
    │    ├── BindLiteralExpression()
    │    ├── BindNameExpression()          → variable resolution
    │    ├── BindAssignmentExpression()    → read-only check + compound expand
    │    ├── BindUnaryExpression()         → operator resolution
    │    ├── BindBinaryExpression()        → operator resolution + constant folding
    │    ├── BindCallExpression()          → function resolution + arg checking
    │    └── BindConversion()              → implicit/explicit conversion
    │
    ▼
Lowerer (高层 BoundTree → 底层 goto-IR)
    │  RewriteIfStatement        → gotoFalse + goto + label
    │  RewriteWhileStatement     → goto + label + gotoTrue
    │  RewriteDoWhileStatement   → label + gotoTrue
    │  RewriteForStatement       → var = lower; while (var <= upper) { body; var++ }
    │  RewriteCompoundAssignment → a = a <op> b
    │  RemoveDeadCode()          → CFG-based unreachable block removal
    │  InsertSequencePoints()    → debug info markers
    │
    ├──▶ Evaluator (树遍历)
    │      EvaluateStatement(block)
    │        → instruction pointer + label dictionary
    │        → switch: Nop/VariableDeclaration/ExpressionStatement/
    │                   Goto/ConditionalGoto/Label/Return
    │      EvaluateExpression(node)
    │        → switch: Literal/Variable/Assignment/Unary/Binary/Call/Conversion
    │
    └──▶ Emitter (Mono.Cecil IL)
           Emit(program, outputPath)
             ├── EmitFunctionDeclaration()  → MethodDefinition per function
             ├── EmitFunctionBody()         → ILProcessor + instructions
             │   ├── EmitStatement()        → switch on BoundNodeKind
             │   └── EmitExpression()       → switch on BoundNodeKind
             └── EmitAssembly()             → write .exe + .pdb
```

### 1.3 当前语言 vs 对标差距

| 维度 | Cocoa 现状 | C# 对标目标 |
|------|-----------|-----------|
| **类型系统** | 5 种：`int`, `bool`, `string`, `any`, `void` | 完整 CLR 类型体系（数值、字符、数组、泛型、可为空等） |

| **互操作** | 仅 `-r` 引用程序集 + 有限内建方法映射 | 调用任意 .NET API、创建对象、实现接口 |
| **工具链** | VS Code 扩展(纯文本正则) | LSP 协议、Roslyn 级语义分析 |

## 2. 总体设计原则

### 2.1 架构决策

| 决策 | 选择 | 理由 |
|------|------|------|
| 类型系统重构 | `TypeSymbol` → 抽象基类 + `TypeKind` 枚举 + 子类层次 | 支持数组/泛型/可为空等参数化类型 |
| 互操作方式 | 编译期元数据解析 + IL emit | 利用现有 Emitter 架构，编译为 .NET 程序集 |
| 语法兼容 | Cocoa 语法向 C# 靠拢但保留特色关键字 | `let`/`var` 保留，`function` 改为可选兼容 |
| 降低策略 | 高层语法 → 底层 goto 的 Lowerer 模式 | 复用现有架构，Evaluator 和 Emitter 只需理解底层 IR |
| 增量建设 | 不破坏现有测试的前提下逐步扩展 | 每个 Phase 增加新测试，旧测试保持绿 |

### 2.3 转换系统重构

```csharp
internal sealed class Conversion
{
    public static readonly Conversion None = new(exists: false, isIdentity: false, isImplicit: false, isExplicit: false);
    public static readonly Conversion Identity = new(exists: true, isIdentity: true, isImplicit: true, isExplicit: false);

    private Conversion(bool exists, bool isIdentity, bool isImplicit, bool isExplicit)
    {
        Exists = exists;
        IsIdentity = isIdentity;
        IsImplicit = isImplicit;
        IsExplicit = isExplicit;
    }

    // 工厂方法用于语义化构造
    public static Conversion Implicit => new(true, false, true, false);
    public static Conversion Explicit => new(true, false, false, true);
    public static Conversion WithUserDefined(Conversion inner, MethodReference? op = null) => ...;

    public bool Exists { get; }
    public bool IsIdentity { get; }
    public bool IsImplicit { get; }
    public bool IsExplicit { get; }

    public static Conversion Classify(TypeSymbol from, TypeSymbol to)
    {
        // 1. Identity
        if (from == to) return Identity;

        // 2. Implicit Numeric Lifting:
        //    byte → short → int → long → float → double → decimal
        if (from.IsNumeric() && to.IsNumeric())
        {
            if (IsImplicitNumericConversion(from.TypeKind, to.TypeKind))
                return Implicit;
            if (IsExplicitNumericConversion(from.TypeKind, to.TypeKind))
                return Explicit;
        }

        // 3. Implicit Nullable: T → T?
        if (to is NullableType nullableTo && nullableTo.UnderlyingType == from)
            return Implicit;

        // 4. Implicit Boxing: valueType → object/any
        if (from.IsValueType() && to.TypeKind == TypeKind.Any)
            return Implicit;

        // 5. Implicit Reference: string → object
        if (from.TypeKind == TypeKind.String && to.TypeKind == TypeKind.Any)
            return Implicit;

        // 6. Explicit Unboxing: object/any → valueType
        if (from.TypeKind == TypeKind.Any && to.IsValueType())
            return Explicit;

        // 7. Explicit: string ↔ int/bool (保留现有)
        if (from.TypeKind == TypeKind.String && to.IsNumeric())
            return Explicit;
        if (from.TypeKind == TypeKind.String && to.TypeKind == TypeKind.Boolean)
            return Explicit;
        if (from.IsNumeric() && to.TypeKind == TypeKind.String)
            return Explicit;
        if (from.TypeKind == TypeKind.Boolean && to.TypeKind == TypeKind.String)
            return Explicit;

        // 8. Explicit: Nullable → T (need null check)
        if (from is NullableType nullableFrom && nullableFrom.UnderlyingType == to)
            return Explicit;

        return None;
    }

    private static bool IsImplicitNumericConversion(TypeKind from, TypeKind to)
    {
        // byte → short → int → long → float → double → decimal (链式)
        var rank = new Dictionary<TypeKind, int>
        {
            [TypeKind.Byte] = 0,
            [TypeKind.Short] = 1,
            [TypeKind.Int32] = 2,
            [TypeKind.Int64] = 3,
            [TypeKind.Float] = 4,
            [TypeKind.Double] = 5,
            [TypeKind.Decimal] = 6,
        };
        return rank.ContainsKey(from) && rank.ContainsKey(to) && rank[from] < rank[to];
    }

    private static bool IsExplicitNumericConversion(TypeKind from, TypeKind to)
    {
        // 逆向：long → int, double → float 等
        var rank = new Dictionary<TypeKind, int> { /* same as above */ };
        return rank.ContainsKey(from) && rank.ContainsKey(to) && rank[from] > rank[to];
    }
}
```

### 3.3 Lowerer 改写方法规范

```csharp
// 改写方法必须完全降低为 goto/label/nop/return 等原始 IR
// Lowerer 继承 BoundTreeRewriter，重写特定 Rewrite* 方法
// 改写结果递归调用 RewriteStatement 以确保完全降低
protected override BoundStatement RewriteSwitchStatement(BoundSwitchStatement node)
{
    // 1. 生成跳转标签
    // 2. 构造 if-goto 链
    // 3. 递归调用 RewriteStatement 以触发进一步的降低
    var endLabel = GenerateLabel();

    // 构建 if-goto 链
    var statements = new List<BoundStatement>();
    foreach (var caseClause in node.Cases)
    {
        var caseLabel = GenerateLabel();
        statements.Add(GotoFalse(node.Syntax, caseLabel,
            Binary(node.Syntax, node.Expression, SyntaxKind.EqualsEqualsToken, caseClause.Value)));
        statements.Add(Goto(node.Syntax, caseLabel));
    }
    if (node.DefaultBody != null)
        statements.AddRange(node.DefaultBody.Statements);
    statements.Add(Label(node.Syntax, node.BreakLabel)); // break

    var result = Block(node.Syntax, statements.ToArray());
    return RewriteStatement(result);  // ← 递归降低
}
```

### 3.4 Evaluator 求值规范

```csharp
// 在 EvaluateStatement 的 switch 或 EvaluateExpression 的 switch 中添加 case
private object? EvaluateStatement(BoundBlockStatement body)
{
    // ... existing code ...
    while (index < statements.Length)
    {
        var statement = statements[index];
        switch (statement.Kind)
        {
            case BoundNodeKind.SwitchStatement:
                EvaluateSwitchStatement((BoundSwitchStatement)statement);
                index++;
                break;
            // ... existing cases ...
        }
    }
}

private void EvaluateSwitchStatement(BoundSwitchStatement node)
{
    var value = EvaluateExpression(node.Expression);
    foreach (var caseClause in node.Cases)
    {
        var caseValue = EvaluateExpression(caseClause.Value);
        if (Equals(value, caseValue))
        {
            EvaluateStatement(caseClause.Body);
            return;
        }
    }
    if (node.DefaultBody != null)
        EvaluateStatement(node.DefaultBody);
}
```

### 3.5 Emitter IL 生成规范

```csharp
// 1. 从 BoundNode 构造 ILProcessor 指令
// 2. 使用 OpCodes 的静态字段引用 IL 指令
// 3. 所有 label fixup 通过 _labels 字典 + _fixups 列表延迟绑定
private void EmitSwitchStatement(ILProcessor ilProcessor, BoundSwitchStatement node)
{
    // 生成 if-goto 链（因为 switch 已在 Lowerer 中降低）
    // 如果 switch 未降低，则手动 emit 跳转表
    foreach (var caseClause in node.Cases)
    {
        EmitExpression(ilProcessor, node.Expression);
        EmitExpression(ilProcessor, caseClause.Value);
        ilProcessor.Emit(OpCodes.Ceq);

        var elseLabel = GenerateLabel();
        _fixups.Add((ilProcessor.Body.Instructions.Count, elseLabel));
        ilProcessor.Emit(OpCodes.Brfalse, Instruction.Create(OpCodes.Nop));

        // emit case body
        foreach (var stmt in caseClause.Body.Statements)
            EmitStatement(ilProcessor, stmt);

        // goto break
        var breakInstr = ilProcessor.Body.Instructions.Count;
        _fixups.Add((breakInstr, node.BreakLabel));
        ilProcessor.Emit(OpCodes.Br, Instruction.Create(OpCodes.Nop));

        // else label target
        _labels.Add(elseLabel, ilProcessor.Body.Instructions.Count);
    }
    // default body
    if (node.DefaultBody != null)
        foreach (var stmt in node.DefaultBody.Statements)
            EmitStatement(ilProcessor, stmt);

    // break label
    _labels.Add(node.BreakLabel, ilProcessor.Body.Instructions.Count);
}
```

```csharp
internal sealed class ProjectConfigParser
{
    public ProjectConfig Parse(string jsonText, string projectDir)
    {
        var root = JsonDocument.Parse(jsonText).RootElement;

        var name = root.TryGetProperty("name", out var n) ? n.GetString()! : Path.GetFileName(projectDir);
        var outputType = root.TryGetProperty("outputType", out var ot) && ot.GetString() == "Library"
            ? OutputKind.Library : OutputKind.ConsoleApplication;
        var sources = root.TryGetProperty("sources", out var s)
            ? s.EnumerateArray().Select(e => e.GetString()!).ToImmutableArray()
            : ImmutableArray.Create("*.co");
        var references = root.TryGetProperty("references", out var r)
            ? r.EnumerateArray().Select(e => e.GetString()!).ToImmutableArray()
            : ImmutableArray<string>.Empty;
        var projectReferences = root.TryGetProperty("projectReferences", out var pr)
            ? pr.EnumerateArray().Select(e => e.GetString()!).ToImmutableArray()
            : ImmutableArray<string>.Empty;
        var packageReferences = root.TryGetProperty("packageReferences", out var pkg)
            ? ParsePackageReferences(pkg)
            : ImmutableDictionary<string, string>.Empty;
        var outputPath = root.TryGetProperty("outputPath", out var op) ? op.GetString()! : ".";
        var moduleName = root.TryGetProperty("moduleName", out var mn) ? mn.GetString() : null;
        var entryPoint = root.TryGetProperty("entryPoint", out var ep) ? ep.GetString() : "Main";
        var debug = !root.TryGetProperty("debug", out var d) || d.GetBoolean();
        var define = root.TryGetProperty("define", out var df)
            ? df.EnumerateArray().Select(e => e.GetString()!).ToImmutableArray()
            : ImmutableArray<string>.Empty;
        var deterministic = !root.TryGetProperty("deterministic", out var det) || det.GetBoolean();
        var treatWarningsAsErrors = root.TryGetProperty("treatWarningsAsErrors", out var twe) && twe.GetBoolean();
        var version = root.TryGetProperty("version", out var ver) ? ver.GetString() : "1.0.0.0";
        var fileVersion = root.TryGetProperty("fileVersion", out var fv) ? fv.GetString() : version;
        var company = root.TryGetProperty("company", out var cmp) ? cmp.GetString() : "";
        var copyright = root.TryGetProperty("copyright", out var cpr) ? cpr.GetString() : "";
        var description = root.TryGetProperty("description", out var desc) ? desc.GetString() : "";

        return new ProjectConfig(name, outputType, projectDir, sources, references,
                                 projectReferences, packageReferences,
                                 outputPath, moduleName, entryPoint, debug,
                                 define, deterministic, treatWarningsAsErrors,
                                 version, fileVersion, company, copyright, description);
    }

    private ImmutableDictionary<string, string> ParsePackageReferences(JsonElement element)
    {
        var dict = ImmutableDictionary.CreateBuilder<string, string>(StringComparer.OrdinalIgnoreCase);
        foreach (var pkg in element.EnumerateObject())
            dict.Add(pkg.Name, pkg.Value.GetString()!);
        return dict.ToImmutable();
    }
}
```

**验证规则**：
- 未知字段静默忽略（允许未来扩展）
- `outputType` 非 `"Exe"`/`"Library"` 时抛出 `ProjectFileFormatException`
- `sources` 中每个模式必须至少匹配一个文件（否则发出警告）
- `references` 路径为空字符串时忽略
- `projectReferences` 中不存在的 .coproj 路径报错
- `packageReferences` 版本号格式非法时发出警告
- `outputPath` 若为相对路径，相对于项目文件所在目录解析
- `version` 格式非 `major.minor[.build[.revision]]` 时发出警告

**验证规则**：
- `projects` 字段为必填，缺失时抛出 `ProjectFileFormatException`
- `projects` 列表空时发出警告
- 每个项目路径转换为绝对路径（相对于 `.cosln` 所在目录）
- 项目路径指向的文件必须存在，否则抛出 `FileNotFoundException`

#### 4.3.6 编译顺序推断

不再依赖数组顺序。编译顺序由 ProjectReference 依赖图拓扑排序决定：

```
推断算法:
  1. 收集所有项目的 projectReferences 引用关系
  2. 构建有向图: A → B (A 引用 B)
  3. 拓扑排序 → 编译顺序（B 先于 A）
  4. 检测环: 若存在循环依赖则报告错误

若不存在 projectReferences（纯独立项目），则按数组顺序编译。
```

#### 4.4.2 核心内部实现

```csharp
internal sealed class ProjectLoader
{
    private readonly DiagnosticBag _diagnostics;
    private readonly NuGetResolver? _nuGetResolver;  // 可选 NuGet 集成

    public ProjectLoader(DiagnosticBag diagnostics)
    {
        _diagnostics = diagnostics;
    }

    public ProjectConfig LoadProject(string projectPath)
    {
        var jsonText = File.ReadAllText(projectPath);
        var projectDir = Path.GetDirectoryName(Path.GetFullPath(projectPath))!;
        return new ProjectConfigParser().Parse(jsonText, projectDir);
    }

    public SolutionConfig LoadSolution(string solutionPath)
    {
        var jsonText = File.ReadAllText(solutionPath);
        var solutionDir = Path.GetDirectoryName(Path.GetFullPath(solutionPath))!;
        return new SolutionConfigParser().Parse(jsonText, solutionDir);
    }

    public ImmutableArray<string> ExpandGlobs(string baseDir, string[] patterns)
    {
        var files = new HashSet<string>(StringComparer.OrdinalIgnoreCase);
        foreach (var pattern in patterns)
        {
            if (pattern.Contains('*') || pattern.Contains('?'))
            {
                var dir = Path.GetDirectoryName(Path.Combine(baseDir, pattern));
                var searchPattern = Path.GetFileName(pattern);
                if (string.IsNullOrEmpty(dir)) dir = baseDir;

                foreach (var file in Directory.EnumerateFiles(dir!, searchPattern, SearchOption.AllDirectories))
                {
                    if (file.EndsWith(".co", StringComparison.OrdinalIgnoreCase))
                        files.Add(Path.GetFullPath(file));
                }
            }
            else
            {
                var fullPath = Path.GetFullPath(Path.Combine(baseDir, pattern));
                if (File.Exists(fullPath))
                    files.Add(fullPath);
                else
                    _diagnostics.ReportProjectFileNotFound(fullPath);
            }
        }
        return files.OrderBy(f => f, StringComparer.OrdinalIgnoreCase).ToImmutableArray();
    }

    public string ComputeOutputPath(ProjectConfig config, string? commandLineOutput)
    {
        if (commandLineOutput != null)
            return Path.GetFullPath(commandLineOutput);

        return Path.GetFullPath(Path.Combine(config.ProjectDir, config.OutputPath));
    }

    public bool ResolveDebugMode(ProjectConfig config, bool? commandLineDebug)
    {
        return commandLineDebug ?? config.Debug;
    }

    public ImmutableArray<SyntaxTree> LoadSyntaxTrees(ProjectConfig config)
    {
        var sourceFiles = ExpandGlobs(config.ProjectDir, config.Sources.ToArray());
        var trees = ImmutableArray.CreateBuilder<SyntaxTree>();
        foreach (var file in sourceFiles)
        {
            var text = SourceText.From(File.ReadAllText(file), Encoding.UTF8);
            var tree = SyntaxTree.Parse(text, _diagnostics, file);
            trees.Add(tree);
        }
        return trees.ToImmutable();
    }

    public ImmutableArray<string> ResolveProjectReferences(ProjectConfig config, SolutionConfig? solution)
    {
        var paths = new HashSet<string>(StringComparer.OrdinalIgnoreCase);
        foreach (var refPath in config.ProjectReferences)
        {
            var absPath = Path.GetFullPath(Path.Combine(config.ProjectDir, refPath));
            if (!File.Exists(absPath))
            {
                _diagnostics.ReportProjectReferenceNotFound(absPath);
                continue;
            }

            // 解析被引用项目的输出路径
            var refConfig = LoadProject(absPath);
            var outputDir = ComputeOutputPath(refConfig, null);
            var outputFile = Path.Combine(outputDir, refConfig.ResolvedModuleName + refConfig.ResolvedExtension);

            if (File.Exists(outputFile))
                paths.Add(outputFile);
            else
                _diagnostics.ReportProjectReferenceOutputMissing(absPath, outputFile);
        }
        return paths.OrderBy(p => p).ToImmutableArray();
    }

    public ImmutableArray<string> ResolvePackageReferences(ProjectConfig config)
    {
        // NuGet 解析策略：
        // 1. 检查项目目录下的 packages/{packageName}/{version}/lib/net9.0/
        // 2. 检查全局 NuGet 缓存 (%USERPROFILE%\.nuget\packages)
        // 3. 检查 NuGet 回退文件夹
        // 4. 可选: 调用 NuGet 的 LockFile 解析（project.assets.json）

        var dlls = new List<string>();
        foreach (var (pkgName, pkgVersion) in config.PackageReferences)
        {
            var searchDirs = new[]
            {
                Path.Combine(config.ProjectDir, "packages", pkgName, pkgVersion, "lib", "net9.0"),
                Path.Combine(
                    Environment.GetFolderPath(Environment.SpecialFolder.UserProfile),
                    ".nuget", "packages", pkgName, pkgVersion, "lib", "net9.0"),
            };

            bool found = false;
            foreach (var dir in searchDirs)
            {
                if (Directory.Exists(dir))
                {
                    var dll = Directory.EnumerateFiles(dir, "*.dll").FirstOrDefault();
                    if (dll != null) { dlls.Add(dll); found = true; break; }
                }
            }
            if (!found)
                _diagnostics.ReportNuGetPackageNotFound(pkgName, pkgVersion);
        }
        return dlls.ToImmutableArray();
    }

    public ImmutableArray<MetadataReference> LoadMetadataReferences(ProjectConfig config, SolutionConfig? solution)
    {
        var refs = ImmutableArray.CreateBuilder<MetadataReference>();

        // 1. 直接 references
        foreach (var r in config.References)
        {
            var path = Path.GetFullPath(Path.Combine(config.ProjectDir, r));
            if (File.Exists(path))
                refs.Add(MetadataReference.CreateFromFile(path));
        }

        // 2. projectReferences
        foreach (var dll in ResolveProjectReferences(config, solution))
            refs.Add(MetadataReference.CreateFromFile(dll));

        // 3. packageReferences
        foreach (var dll in ResolvePackageReferences(config))
            refs.Add(MetadataReference.CreateFromFile(dll));

        return refs.ToImmutable();
    }

    public ImmutableArray<string> TopologicalSort(SolutionConfig solution, IEnumerable<ProjectConfig> projects)
    {
        // Kahn 算法 (BFS 拓扑排序)
        var projectMap = projects.ToDictionary(p => Path.GetFullPath(p.ProjectDir), p => p);
        var inDegree = projectMap.Keys.ToDictionary(k => k, _ => 0);
        var adjacency = projectMap.Keys.ToDictionary(k => k, _ => new List<string>());

        foreach (var (path, config) in projectMap)
        {
            foreach (var refRelative in config.ProjectReferences)
            {
                var refAbsolute = Path.GetFullPath(Path.Combine(config.ProjectDir, refRelative));
                if (adjacency.ContainsKey(refAbsolute))
                {
                    adjacency[refAbsolute].Add(path);
                    inDegree[path]++;
                }
            }
        }

        var queue = new Queue<string>(inDegree.Where(kv => kv.Value == 0).Select(kv => kv.Key));
        var sorted = new List<string>();

        while (queue.Count > 0)
        {
            var node = queue.Dequeue();
            sorted.Add(node);
            foreach (var neighbor in adjacency[node])
            {
                if (--inDegree[neighbor] == 0)
                    queue.Enqueue(neighbor);
            }
        }

        if (sorted.Count != projectMap.Count)
            _diagnostics.ReportCircularDependency();

        return sorted.Select(s => projectMap[s].ProjectDir).ToImmutableArray();
    }

    public bool DetectCycles(SolutionConfig solution, out ImmutableArray<string> cyclePath)
    {
        // DFS 检测环
        var visited = new HashSet<string>();
        var inStack = new HashSet<string>();
        var stack = new List<string>();
        // ... 标准 DFS 环检测实现
        cyclePath = ImmutableArray<string>.Empty;
        return false;
    }
}
```

| `.cosln` 文件不存在 | `FileNotFoundException` |
| JSON 解析失败（非法格式） | `ProjectFileFormatException` + 行号 |
| 缺少必填字段 `projects`（cosln） | `ProjectFileFormatException` |
| `outputType` 非法值 | `ProjectFileFormatException` |

| `version` 格式非法 | `DiagnosticBag.Warning` + 回退 `"1.0.0.0"` |

### 4.5 Glob 展开规则

#### 4.5.1 模式语法

| 模式 | 匹配 | 内部实现 |
|------|------|----------|
| `*.co` | 当前目录所有 `.co` 文件 | `Directory.EnumerateFiles(dir, "*.co")` + 单层搜索 |
| `src/**/*.co` | `src/` 及其子目录下所有 `.co` 文件 | `Directory.EnumerateFiles(dir, "*.co", AllDirectories)` 后过滤前缀 |
| `src/*.co` | `src/` 下一层所有 `.co` 文件 | `Directory.EnumerateFiles(dir, "*.co", TopDirectoryOnly)` |
| `main.co` | 单文件 | `File.Exists` 检查，精确匹配 |
| `["a.co", "b.co"]` | 多个指定文件 | 分别应用单文件规则，结果取并集 |

#### 4.5.2 `**` 递归处理策略

目录分隔符统一使用 `/`（跨平台），在 Windows 上 `Path.DirectorySeparatorChar` 自动转换。

```
模式: src/**/*.co
分解:
  1. 提取固定前缀: "src"
  2. 提取文件名模式: "*.co"
  3. 搜索 src 目录下所有 *.co（递归）
  4. 过滤：仅保留 src/ 为前缀的文件
```

#### 4.5.3 去重与排序

- 同一文件通过多个模式匹配时自动去重（`HashSet<string>` + `OrdinalIgnoreCase`）
- 最终列表按路径字母序排列，确保编译顺序可复现

### 5.1 `switch` 语句

**语法定义**：
```
switch-statement:
    'switch' '(' expression ')' '{' switch-sections? '}'

switch-sections:
    switch-section
    switch-sections switch-section

switch-section:
    'case' constant-expression ':' '{' statement* 'break' '}'
    'default' ':' '{' statement* 'break' '}'
```

> 每个 `case` 体内最后一条语句必须显式 `break`，不支持 fall-through。

**Lexer 变更**：
```csharp
// SyntaxKind.cs 新增
SwitchKeyword,     // "switch"
CaseKeyword,       // "case"
DefaultKeyword,    // "default"

// 节点类型
SwitchStatement,
SwitchCase,
```

**Parser 实现**：
```csharp
// Parser.cs 新增 ParseSwitchStatement，在 ParseStatement 中 case
private StatementSyntax ParseSwitchStatement()
{
    var keyword = MatchToken(SyntaxKind.SwitchKeyword);
    MatchToken(SyntaxKind.OpenParenthesisToken);
    var expression = ParseExpression();
    MatchToken(SyntaxKind.CloseParenthesisToken);
    var openBrace = MatchToken(SyntaxKind.OpenBraceToken);

    var cases = ImmutableArray.CreateBuilder<SwitchCaseSyntax>();
    ElseClauseSyntax? defaultClause = null;

    while (Current.Kind != SyntaxKind.CloseBraceToken &&
           Current.Kind != SyntaxKind.EndOfFileToken)
    {
        if (Current.Kind == SyntaxKind.CaseKeyword)
        {
            var caseKeyword = NextToken();
            var value = ParseExpression();
            var colon = MatchToken(SyntaxKind.ColonToken);

            var statements = ImmutableArray.CreateBuilder<StatementSyntax>();
            while (Current.Kind != SyntaxKind.CaseKeyword &&
                   Current.Kind != SyntaxKind.DefaultKeyword &&
                   Current.Kind != SyntaxKind.CloseBraceToken &&
                   Current.Kind != SyntaxKind.EndOfFileToken)
            {
                statements.Add(ParseStatement());
            }

            cases.Add(new SwitchCaseSyntax(_syntaxTree, caseKeyword, value, colon, statements.ToImmutable()));
        }
        else if (Current.Kind == SyntaxKind.DefaultKeyword)
        {
            var defaultKeyword = NextToken();
            var colon = MatchToken(SyntaxKind.ColonToken);

            var statements = ImmutableArray.CreateBuilder<StatementSyntax>();
            while (Current.Kind != SyntaxKind.CaseKeyword &&
                   Current.Kind != SyntaxKind.DefaultKeyword &&
                   Current.Kind != SyntaxKind.CloseBraceToken &&
                   Current.Kind != SyntaxKind.EndOfFileToken)
            {
                statements.Add(ParseStatement());
            }

            // 用 ElseClauseSyntax 复用，语义映射 "default" = "else"
            defaultClause = new ElseClauseSyntax(_syntaxTree, defaultKeyword,
                new BlockStatementSyntax(_syntaxTree, SyntaxFactory.MissingToken(SyntaxKind.OpenBraceToken),
                    statements.ToImmutable(), SyntaxFactory.MissingToken(SyntaxKind.CloseBraceToken)));
        }
        else
        {
            // 跳过高危 Token 防无限循环
            NextToken();
        }
    }

    var closeBrace = MatchToken(SyntaxKind.CloseBraceToken);
    return new SwitchStatementSyntax(_syntaxTree, keyword, expression, openBrace,
        cases.ToImmutable(), defaultClause, closeBrace);
}
```

**Lowerer 降低算法**（if-goto 链）：
```csharp
// switch (x) { case 1: body1; case 2: body2; default: body3; }
//                                       ↓
// if x == 1 goto case1
// if x == 2 goto case2
// goto default
// case1: body1; goto break;
// case2: body2; goto break;
// default: body3;
// break:

protected override BoundStatement RewriteSwitchStatement(BoundSwitchStatement node)
{
    var nextLabel = GenerateLabel();
    var statements = new List<BoundStatement>();

    foreach (var caseClause in node.Cases)
    {
        var caseLabel = GenerateLabel();
        var eq = Binary(node.Syntax, node.Expression,
            SyntaxKind.EqualsEqualsToken, caseClause.Value);
        statements.Add(GotoFalse(node.Syntax, caseLabel, eq));
        statements.Add(Goto(node.Syntax, caseLabel));
    }

    // 全部 case 不匹配 → default 或 break
    if (node.DefaultBody != null)
    {
        statements.AddRange(node.DefaultBody.Statements);
    }
    statements.Add(Goto(node.Syntax, node.BreakLabel));

    // 插入 case 标签 + body
    for (int i = 0; i < node.Cases.Length; i++)
    {
        var caseLabel = GenerateLabel();
        statements.Add(Label(node.Syntax, caseLabel));
        statements.AddRange(node.Cases[i].Body.Statements);
        statements.Add(Goto(node.Syntax, node.BreakLabel));
    }

    return RewriteStatement(Block(node.Syntax, statements.ToArray()));
}
```

**Emitter**：switch 已在 Lowerer 中降低为 goto 链，Emitter 无需特殊处理。除非保留 BoundSwitchStatement 用于跳转表优化。

```csharp
// foreach (x in arr)  body
//   ↓  当 arr 类型为 ArrayType 时
// var i = 0
// while i < arr.Length
// {
//     let x = arr[i]
//     body
//     continue:
//     i = i + 1
// }
// break:
protected override BoundStatement RewriteForeachStatement(BoundForeachStatement node)
{
    if (node.Expression.Type is ArrayType arrayType)
    {
        var iVariable = new LocalVariableSymbol("$i", false, TypeSymbol.Int32, null);
        var iInit = new BoundLiteralExpression(node.Syntax, 0);
        var iDecl = new BoundVariableDeclaration(node.Syntax, iVariable, iInit);

        var lengthExpr = new BoundMemberAccessExpression(
            node.Syntax, node.Expression, "Length",
            TypeSymbol.Int32, isStatic: false);
        var condition = new BoundBinaryExpression(node.Syntax,
            Variable(node.Syntax, iVariable),
            BoundBinaryOperator.Bind(SyntaxKind.LessToken, TypeSymbol.Int32, TypeSymbol.Int32)!,
            lengthExpr);

        var xDecl = new BoundVariableDeclaration(node.Syntax, node.Variable,
            new BoundArrayAccessExpression(node.Syntax, node.Expression, Variable(node.Syntax, iVariable)));

        var increment = new BoundAssignmentExpression(node.Syntax, iVariable,
            new BoundBinaryExpression(node.Syntax,
                Variable(node.Syntax, iVariable),
                BoundBinaryOperator.Bind(SyntaxKind.PlusToken, TypeSymbol.Int32, TypeSymbol.Int32)!,
                new BoundLiteralExpression(node.Syntax, 1)));

        var whileBody = new BoundBlockStatement(node.Syntax,
            ImmutableArray.Create<BoundStatement>(
                xDecl,
                node.Body,
                new BoundLabelStatement(node.Syntax, node.ContinueLabel),
                new BoundExpressionStatement(node.Syntax, increment)));

        var whileStmt = new BoundWhileStatement(node.Syntax, condition, whileBody,
            node.BreakLabel, GenerateLabel());

        return RewriteStatement(new BoundBlockStatement(node.Syntax,
            ImmutableArray.Create<BoundStatement>(iDecl, whileStmt)));
    }

    // IEnumerable 路径（需 try/finally Phase 3）
    throw new NotImplementedException("foreach on IEnumerable requires Phase 3 (try/finally + IDisposable)");
}
```

```csharp
private BoundStatement BindForeachStatement(ForeachStatementSyntax syntax)
{
    var expression = BindExpression(syntax.Expression);
    var variableType = TypeSymbol.Int32; // 默认从集合元素类型推断

    if (expression.Type is ArrayType arrayType)
        variableType = arrayType.ElementType;
    else if (expression.Type.TypeKind == TypeKind.String)

    _scope.TryDeclareVariable(variable);
    var body = BindLoopBody(syntax.Body, out var breakLabel, out var continueLabel);
    _scope = _scope.Parent!;

    return new BoundForeachStatement(syntax, variable, expression, body, breakLabel, continueLabel);
}
```

### 5.4 `++`/`--` 自增自减

**Token**：`PlusPlusToken`（`++`）, `MinusMinusToken`（`--`）

**Lexer 最长匹配算法**：
```csharp
case '+':
    _position++;
    if (Current == '+')
    {
        _kind = SyntaxKind.PlusPlusToken;
        _position++;
    }
    else if (Current == '=')
    {
        _kind = SyntaxKind.PlusEqualsToken;
        _position++;
    }
    else
        _kind = SyntaxKind.PlusToken;
    break;
```

**Parser — 后缀形式**（在 `ParsePrimaryExpression` 中检查）：
```csharp
private ExpressionSyntax ParsePrimaryExpression()
{
    ExpressionSyntax expr;
    switch (Current.Kind)
    {
        // ... existing cases ...
        default:
            expr = ParseNameOrCallExpression();
            break;
    }

    // Postfix operators: ++, --
    while (true)
    {
        if (Current.Kind == SyntaxKind.PlusPlusToken)
        {
            var op = NextToken();
            expr = new PostfixUnaryExpressionSyntax(_syntaxTree, expr, op);
        }
        else if (Current.Kind == SyntaxKind.MinusMinusToken)
        {
            var op = NextToken();
            expr = new PostfixUnaryExpressionSyntax(_syntaxTree, expr, op);
        }
        else
            break;
    }
    return expr;
}
```

**Parser — 前缀形式**（在 `ParseBinaryExpression` 的一元运算符部分检查）：
```csharp
case SyntaxKind.PlusPlusToken:
case SyntaxKind.MinusMinusToken:

    left = new PrefixUnaryExpressionSyntax(_syntaxTree, operatorToken, operand);
    break;
```

```csharp
private BoundExpression BindTernaryExpression(TernaryExpressionSyntax syntax)
{
    var condition = BindExpression(syntax.Condition, TypeSymbol.Boolean);
    var whenTrue = BindExpression(syntax.WhenTrue);
    var whenFalse = BindExpression(syntax.WhenFalse);

    // 类型统一：whenTrue 和 whenFalse 必须互相可转换
    var resultType = whenTrue.Type;
    var conversion = Conversion.Classify(whenFalse.Type, resultType);
    if (!conversion.Exists)
    {
        // 尝试反向转换
        conversion = Conversion.Classify(resultType, whenFalse.Type);
        if (conversion.Exists)
            resultType = whenFalse.Type;
        else
        {
            _diagnostics.ReportCannotConvert(syntax.WhenTrue.Location, whenTrue.Type, whenFalse.Type);
            return new BoundErrorExpression(syntax);
        }
    }
    // 应用转换
    var convertedTrue = BindConversion(syntax.WhenTrue.Location, whenTrue, resultType);
    var convertedFalse = BindConversion(syntax.WhenFalse.Location, whenFalse, resultType);

    return new BoundTernaryExpression(syntax, condition, convertedTrue, convertedFalse);
}
```

```csharp
//     result = a

// else
// {
//     result = b
// }

protected override BoundStatement RewriteExpressionStatement(BoundExpressionStatement node)
{
    if (node.Expression is BoundTernaryExpression ternary)
    {
        var tempVar = new LocalVariableSymbol("$temp", false, ternary.Type, null);
        var resultDecl = new BoundVariableDeclaration(node.Syntax, tempVar,
            new BoundLiteralExpression(node.Syntax, GetDefaultValue(ternary.Type)));
        var thenAssign = new BoundAssignmentExpression(node.Syntax, tempVar, ternary.WhenTrue);
        var elseAssign = new BoundAssignmentExpression(node.Syntax, tempVar, ternary.WhenFalse);
        var ifStmt = new BoundIfStatement(node.Syntax, ternary.Condition,

        return RewriteStatement(new BoundBlockStatement(node.Syntax,
            ImmutableArray.Create<BoundStatement>(resultDecl, ifStmt)));

    return base.RewriteExpressionStatement(node);
}
```

- 嵌套三元 `a ? b : c ? d : e` → 右结合（由 Parser 递归控制）

### 5.7 `char` 类型 + 字符字面量

**Lexer**：
```csharp
case '\'':
    ReadCharLiteral();
    break;

private void ReadCharLiteral()
{
    _position++; // 跳过 '
    if (Current == '\0' || Current == '\r' || Current == '\n')
    {
        _diagnostics.ReportUnterminatedString(Current.Location);
        _kind = SyntaxKind.BadToken;
        return;
    }

    char value;
    if (Current == '\\')
    {
        _position++;
        value = Current switch
        {
            'n' => '\n', 't' => '\t', 'r' => '\r',
            '0' => '\0', '\\' => '\\', '\'' => '\'',
            _ => { _diagnostics.ReportBadEscape(Current.Location); Current; }
        };
        _position++;
    }
    else
    {
        value = Current;
        _position++;
    }

    if (Current != '\'')
    {
        _diagnostics.ReportUnterminatedString(Current.Location);
    }
    else
        _position++; // 跳过 '

    _kind = SyntaxKind.CharToken;
    _value = value;
}
```

**Evaluator**：char 作为数值参与运算（int 升级），字符串拼接时转为 string。

**Emitter**：`char` 对应 `OpCodes.Ldc_I4_S`（加载 char 值作为 int），需要装箱时 `Box<char>`。

### 5.8 `float`/`double`/`decimal` 数值类型

**Lexer** — 数字字面量扩展到浮点：
```csharp
private void ReadNumber()
{
    var isReal = false;
    while (char.IsDigit(Current))
        _position++;

    if (Current == '.' && char.IsDigit(Lookahead))
    {
        isReal = true;
        _position++;
        while (char.IsDigit(Current))
            _position++;
    }

    var text = _text.ToString(_start, _position - _start);

    if (Current == 'f' || Current == 'F')
    {
        _position++;
        if (float.TryParse(text, out var f))
            _value = f;
        _kind = SyntaxKind.FloatToken;
    }
    else if (Current == 'm' || Current == 'M')
    {
        _position++;
        if (decimal.TryParse(text, out var d))
            _value = d;
        _kind = SyntaxKind.DecimalToken;
    }
    else if (isReal || Current == 'd' || Current == 'D')
    {
        if (Current == 'd' || Current == 'D') _position++;
        if (double.TryParse(text, out var d))
            _value = d;
        _kind = SyntaxKind.DoubleToken;
    }
    else
    {
        if (Current == 'L' || Current == 'l')
        {
            _position++;
            if (long.TryParse(text, out var l))
                _value = l;
            _kind = SyntaxKind.LongToken;
        }
        else
        {
            if (int.TryParse(text, out var value))
                _value = value;
            else
                _diagnostics.ReportInvalidNumber(...);
            _kind = SyntaxKind.NumberToken;
        }
    }
}
```

**运算符表扩展**：BoundBinaryOperator 需要为每个数值类型组合添加条目：

### 5.10 C 风格 `for (init; cond; update)` 循环

**语法定义**：
```
for-statement:
    'for' '(' for-initializer? ';' for-condition? ';' for-update? ')' statement

for-initializer:
    variable-declaration
    expression-statement

for-condition:
    expression

for-update:
    expression
```

**与传统 Cocoa `for-range` 对比**：

| 特性 | `for i = low to high` (✅) | `for (init; cond; update)` (🔧) |
|------|---------------------------|-------------------------------|
| 循环变量 | 不可变，自动递增 | 可变，手动控制 |
| 迭代控制 | 范围 bound | 任意条件表达式 |
| 灵活性 | 仅递增步长 1 | 任意更新（递增/递减/其他） |
| 适用场景 | 简单范围迭代 | 复杂循环逻辑 |

**Binder 处理**：引入新的 BoundNode `BoundForStatement`（不同于 `for-range` 的 BoundForStatement）。

```csharp

    public BoundExpression? Condition { get; }
    public BoundExpression? Update { get; }
    public BoundStatement Body { get; }
```

**Lowerer 降低**：
```
for (init; cond; update) body

  ↓ 降低为

{
    init
    :label_continue:
    if (!cond) goto label_break
    body
    update
    goto label_continue
    :label_break:
}
```

各部分省略时的处理：
- `init` 省略 → 无初始化
- `cond` 省略 → 视为 `true`（无限循环，需 `break` 退出）
- `update` 省略 → 无更新表达式

**Emitter**：使用 `Br`/`Brfalse`/`Brtrue` + labels，与 existing lowering-to-gotos 一致。

**Parser 实现**：
```csharp
private StatementSyntax ParseCForStatement()
{
    var keyword = MatchToken(SyntaxKind.ForKeyword);
    var openParen = MatchToken(SyntaxKind.OpenParenthesisToken);

    // 初始化部分
    VariableDeclarationSyntax? initVar = null;
    ExpressionSyntax? initExpr = null;
    if (Current.Kind != SyntaxKind.SemicolonToken)
    {
        if (Current.Kind == SyntaxKind.VarKeyword)
            initVar = ParseVariableDeclaration(); // 只能用 var（变量可变），不能用 let
        else
            initExpr = ParseExpression();
    }
    MatchToken(SyntaxKind.SemicolonToken);

    // 条件部分
    ExpressionSyntax? condition = null;
    if (Current.Kind != SyntaxKind.SemicolonToken)
        condition = ParseExpression();
    MatchToken(SyntaxKind.SemicolonToken);

    // 更新部分
    ExpressionSyntax? update = null;
    if (Current.Kind != SyntaxKind.CloseParenthesisToken)
        update = ParseExpression();
    MatchToken(SyntaxKind.CloseParenthesisToken);

    var body = ParseStatement();

    return new CForStatementSyntax(_syntaxTree, keyword, openParen, initVar, initExpr,
        condition, update, body);
}
```

**Cocoa 示例**：
```

{
    print(i)
}

// 递减
for (var i = 9; i >= 0; i--)
{
    print(i)
}

// 省略 init
var i = 0
for (; i < 10; i++)
{
    print(i)
}

{
    if (done)
    {
        break
    }
}
```

**依赖关系**：依赖 4.4 (`++`/`--` 自增自减) 和 4.5 (`;` 分号作为 for 中分隔符)。

### 5.11 `for-range` 括号与 `var`/`let` 可选增强

对 Cocoa `for-range` 循环增加 C# 风格括号形式（替换原生无括号风格）：

```
// C# 风格（推荐）：括号包裹 range，var 声明可变变量
for (var i = 0 to 10) { body }

// 省略 var（变量需已在作用域内）
for (i = 0 to 10) { body }
```

> **注意**：只能用 `var`，不能用 `let`。循环变量 `i` 在每次迭代后递增，是可变的。省略 `var` 时，`i` 必须在外部作用域已定义，否则编译器报错。

**Parser 变更**：修改 `ParseForStatement`，在遇到 `(` 时先尝试消费括号再解析 range 部分。

```csharp
private StatementSyntax ParseForStatement()
{
    var keyword = MatchToken(SyntaxKind.ForKeyword);

    // 检测括号形式：for (var i = 0 to 10) 或 for (i = 0 to 10)
    bool hasParens = Current.Kind == SyntaxKind.OpenParenthesisToken;
    if (hasParens) NextToken(); // 消费 '('

    // 解析可选的 var 声明
    VariableDeclarationSyntax? variable = null;
    SyntaxToken? identifier;
    if (Current.Kind == SyntaxKind.VarKeyword)
        variable = ParseVariableDeclaration();
    else
        identifier = MatchToken(SyntaxKind.IdentifierToken);

    var equalsToken = MatchToken(SyntaxKind.EqualsToken);
    var lowerBound = ParseExpression();
    var toKeyword = MatchToken(SyntaxKind.ToKeyword);
    var upperBound = ParseExpression();

    if (hasParens) MatchToken(SyntaxKind.CloseParenthesisToken); // 消费 ')'

    var body = ParseStatement();

    if (variable != null)
        return new ForStatementSyntax(_syntaxTree, keyword, variable.Identifier,
            variable.EqualsToken, lowerBound, toKeyword, upperBound, body);
    else
        return new ForStatementSyntax(_syntaxTree, keyword, identifier.Value,
            equalsToken, lowerBound, toKeyword, upperBound, body);
}
```

**Binder/Lowerer/Emitter**：无需改动，语法树结构完全相同（仅解析路径不同）。

```csharp
private TypeClauseSyntax ParseTypeClause()
{
    var colonToken = MatchToken(SyntaxKind.ColonToken);
    var identifier = MatchToken(SyntaxKind.IdentifierToken);

    // 读取后面的方括号：int[], int[,], int[][]
    var arrayRanks = ImmutableArray.CreateBuilder<int>();

        while (Current.Kind == SyntaxKind.CommaToken)
        {
            rank++;
            NextToken();
        }
        MatchToken(SyntaxKind.CloseBracketToken);

    }

    return new TypeClauseSyntax(_syntaxTree, colonToken, identifier,
        arrayRanks.ToImmutable());
}
```

```csharp
private ExpressionSyntax ParsePrimaryExpression()
{
    ExpressionSyntax expr;
    switch (Current.Kind)
    {
        case SyntaxKind.OpenParenthesisToken:
            expr = ParseParenthesizedExpression();
            break;
        case SyntaxKind.FalseKeyword:
        case SyntaxKind.TrueKeyword:
            expr = ParseBooleanLiteral();
            break;
        case SyntaxKind.NumberToken:
        case SyntaxKind.FloatToken:
        case SyntaxKind.DoubleToken:
        case SyntaxKind.DecimalToken:
        case SyntaxKind.LongToken:
            expr = ParseNumberLiteral();
            break;
        case SyntaxKind.StringToken:
            expr = ParseStringLiteral();
            break;
        case SyntaxKind.CharToken:
            expr = ParseCharLiteral();
            break;
        case SyntaxKind.NullKeyword:
            expr = ParseNullLiteral();
            break;
        case SyntaxKind.NewKeyword:
            expr = ParseArrayCreationExpression();  // 或 NewExpression
            break;
        default:
            expr = ParseNameOrCallExpression();
            break;
    }

    // 后缀运算符循环：. , [ ], ++, --
    while (true)
    {
        if (Current.Kind == SyntaxKind.DotToken)
        {
            var dot = NextToken();
            var name = MatchToken(SyntaxKind.IdentifierToken);

            if (Current.Kind == SyntaxKind.OpenParenthesisToken)
            {
                // obj.Method(args) — 整合为 CallExpressionSyntax 的成员访问形式
                var openParen = NextToken();
                var args = ParseArguments();
                var closeParen = MatchToken(SyntaxKind.CloseParenthesisToken);
                expr = new CallExpressionSyntax(_syntaxTree, expr, /* is member */ true, dot,
                    name, openParen, args, closeParen);
            }
            else
            {
                expr = new MemberAccessExpressionSyntax(_syntaxTree, expr, dot, name);
            }
        }
        else if (Current.Kind == SyntaxKind.OpenBracketToken)
        {
            // arr[index]
            var openBracket = NextToken();
            var index = ParseExpression();
            var closeBracket = MatchToken(SyntaxKind.CloseBracketToken);
            expr = new ArrayAccessExpressionSyntax(_syntaxTree, expr, openBracket, index, closeBracket);
        }
        else if (Current.Kind == SyntaxKind.PlusPlusToken)
        {
            expr = new PostfixUnaryExpressionSyntax(_syntaxTree, expr, NextToken());
        }
        else if (Current.Kind == SyntaxKind.MinusMinusToken)
        {
            expr = new PostfixUnaryExpressionSyntax(_syntaxTree, expr, NextToken());
        }
        else
            break;
    }
    return expr;
}
```

```csharp
internal sealed class MetadataResolver
{
    // ===== 缓存系统 =====
    private readonly Dictionary<string, AssemblyDefinition> _assemblyCache;
    private readonly Dictionary<(string AssemblyName, string TypeFullName), TypeDefinition> _typeCache;
    private readonly Dictionary<TypeSymbol, TypeReference> _cocoaToClrMap; // 双向映射

    // ===== 内置类型映射表 =====
    private static readonly Dictionary<TypeKind, (string ClrName, PrimitiveType Type)> _primitiveMap = new()
    {
        [TypeKind.Any]     = ("System.Object",   TypeSymbol.Any),
        [TypeKind.Boolean] = ("System.Boolean",  TypeSymbol.Boolean),
        [TypeKind.Int32]   = ("System.Int32",    TypeSymbol.Int32),
        [TypeKind.Int64]   = ("System.Int64",    TypeSymbol.Int64),
        [TypeKind.Float]   = ("System.Single",   TypeSymbol.Float),
        [TypeKind.Double]  = ("System.Double",   TypeSymbol.Double),
        [TypeKind.Decimal] = ("System.Decimal",  TypeSymbol.Decimal),
        [TypeKind.Char]    = ("System.Char",     TypeSymbol.Char),
        [TypeKind.Byte]    = ("System.Byte",     TypeSymbol.Byte),
        [TypeKind.Short]   = ("System.Int16",    TypeSymbol.Short),
        [TypeKind.String]  = ("System.String",   TypeSymbol.String),
        [TypeKind.Void]    = ("System.Void",     TypeSymbol.Void),
    };

    // ===== 主要 API =====
    public MetadataResolver(string[] referencePaths)
    {
        _assemblyCache = new Dictionary<string, AssemblyDefinition>();
        _typeCache = new Dictionary<(string, string), TypeDefinition>();
        _cocoaToClrMap = new Dictionary<TypeSymbol, TypeReference>();
        _clrToCocoaMap = new Dictionary<TypeReference, TypeSymbol>();

        LoadAssemblies(referencePaths);
        BuildPrimitiveTypeMap();
    }

    /// <summary>根据类型全名解析为 Cocoa TypeSymbol</summary>
    public TypeSymbol? ResolveType(string fullName, ImmutableArray<TypeSymbol>? genericArgs = null)
    {
        // 1. 检查是否为基元类型
        var primitive = _primitiveMap.Values.FirstOrDefault(p => p.ClrName == fullName);
        if (primitive.Type != null) return primitive.Type;

        // 2. 在所有缓存程序集中搜索 TypeDefinition
        foreach (var (asmName, asm) in _assemblyCache)
        {
            var key = (asmName, fullName);
            if (!_typeCache.TryGetValue(key, out var typeDef))
            {
                typeDef = asm.MainModule.GetType(fullName);
                if (typeDef != null)
                    _typeCache[key] = typeDef;
            }

            if (typeDef != null)
            {
                // 3. 根据是否有泛型参数决定返回类型
                if (genericArgs.HasValue && genericArgs.Value.Length > 0)
                {
                    var genericDef = new GenericTypeDefinition(fullName,
                        typeDef.GenericParameters.Select(p => p.Name).ToImmutableArray());
                    return genericDef.Construct(genericArgs.Value);
                }

                // 4. 检查是否为枚举
                if (typeDef.IsEnum)
                {
                    var underlyingType = MapType(typeDef.Fields.First(f => f.Name == "value__").FieldType);
                    var members = new Dictionary<string, object>();
                    foreach (var field in typeDef.Fields.Where(f => f.IsStatic))
                        members[field.Name] = field.Constant;
                    return new EnumType(fullName, underlyingType, members.ToImmutableDictionary());
                }

                // 5. 返回一般类型引用
                return new ExternalType(fullName, this, typeDef);
            }
        }

        return null; // 未找到

        // ↑
        // └── 命名空间感知的调用路径：
        //     Binder 先调用 ImportTable.ResolveType("Console", this)
        //     → ImportTable 依次尝试: "System.Console", "System.Collections.Generic.Console", ...
        //     → 每次尝试均调用此 ResolveType(fullName)
        //     → "System.Console" 命中，返回 ExternalType("System.Console")
    }

    /// <summary>解析静态方法</summary>
    public MethodReference? ResolveStaticMethod(
        TypeSymbol type, string methodName, TypeSymbol[] paramTypes)
    {
        var typeDef = GetTypeDefinition(type);
        if (typeDef == null) return null;

        var methods = typeDef.Methods
            .Where(m => m.IsStatic && m.Name == methodName);

        return SelectBestOverload(type, methods, paramTypes);
    }

    /// <summary>解析实例方法</summary>
    public MethodReference? ResolveInstanceMethod(
        TypeSymbol type, string methodName, TypeSymbol[] paramTypes)
    {
        var typeDef = GetTypeDefinition(type);
        if (typeDef == null) return null;

        var methods = typeDef.Methods
            .Where(m => !m.IsStatic && m.Name == methodName);

        return SelectBestOverload(type, methods, paramTypes);
    }

    /// <summary>解析构造函数</summary>
    public MethodReference? ResolveConstructor(TypeSymbol type, TypeSymbol[] paramTypes)
    {
        var typeDef = GetTypeDefinition(type);
        if (typeDef == null) return null;

        return SelectBestOverload(type,
            typeDef.Methods.Where(m => m.IsConstructor && !m.IsStatic), paramTypes);
    }

    /// <summary>解析属性</summary>
    public PropertyDefinition? ResolveProperty(TypeSymbol type, string propertyName)
    {
        var typeDef = GetTypeDefinition(type);
        return typeDef?.Properties.FirstOrDefault(p => p.Name == propertyName);
    }

    /// <summary>解析字段</summary>
    public FieldDefinition? ResolveField(TypeSymbol type, string fieldName)
    {
        var typeDef = GetTypeDefinition(type);
        return typeDef?.Fields.FirstOrDefault(f => !f.IsStatic && f.Name == fieldName);
    }

    // ===== 私有方法 =====

    private void LoadAssemblies(string[] paths)
    {
        foreach (var path in paths)
        {
            if (!_assemblyCache.ContainsKey(path))
            {
                var asm = AssemblyDefinition.ReadAssembly(path);
                _assemblyCache[path] = asm;
                _assemblyCache[asm.Name.Name] = asm;
            }
        }
    }

    private void BuildPrimitiveTypeMap()
    {
        foreach (var (kind, (clrName, cocoaType)) in _primitiveMap)
        {
            foreach (var asm in _assemblyCache.Values)
            {
                var typeDef = asm.MainModule.GetType(clrName);
                if (typeDef != null)
                {
                    var typeRef = _assemblyCache.Values
                        .SelectMany(a => a.Modules)
                        .SelectMany(m => m.Types)
                        .FirstOrDefault(t => t.FullName == clrName);
                    if (typeRef != null)
                    {
                        var imported = _assemblyDefinition.MainModule.ImportReference(typeRef);
                        _cocoaToClrMap[cocoaType] = imported;
                        _clrToCocoaMap[imported] = cocoaType;
                    }
                    break;
                }
            }
        }
    }

    private MethodReference? SelectBestOverload(
        TypeSymbol type, IEnumerable<MethodDefinition> methods, TypeSymbol[] paramTypes)
    {
        foreach (var method in methods)
        {
            if (method.Parameters.Count != paramTypes.Length)
                continue;

            bool match = true;
            for (int i = 0; i < paramTypes.Length; i++)
            {
                var paramClrType = method.Parameters[i].ParameterType;
                var paramCocoaType = MapType(paramClrType);
                var conversion = Conversion.Classify(paramTypes[i], paramCocoaType);
                if (!conversion.IsImplicit && !conversion.IsIdentity)
                {
                    match = false;
                    break;
                }
            }

            if (match)
            {
                var imported = _assemblyDefinition.MainModule.ImportReference(method);
                return imported;
            }
        }
        return null;
    }

    public TypeReference GetTypeReference(TypeSymbol type)
    {
        if (_cocoaToClrMap.TryGetValue(type, out var cached))
            return cached;

        if (type is ArrayType arrayType)
        {
            var elementRef = GetTypeReference(arrayType.ElementType);
            return new ArrayType(elementRef, arrayType.Rank);
        }

        if (type is ConstructedGenericType generic)
        {
            var defRef = GetTypeReference(generic.Definition) as TypeDefinition;
            var instance = new GenericInstanceType(defRef);
            foreach (var arg in generic.TypeArguments)
                instance.GenericArguments.Add(GetTypeReference(arg));
            return _assemblyDefinition.MainModule.ImportReference(instance);
        }

        throw new NotSupportedException($"Type '{type}' cannot be mapped to a CLR reference.");
    }
}
```

### 7.2 完整的互操作调用流程

以 `System.Console.WriteLine("hello")` 为例：

```
Phase 1: Parser
────────────────────────────────────────────────────────────
ParsePrimaryExpression:
  Identifier("System") → NameExpression
  while(Current == DotToken):
    . Console → MemberAccessExpression(System, ., Console)
    . WriteLine → MemberAccessExpression(..., ., WriteLine)
    if (Current == OpenParenToken):
      CallExpression(expr=MemberAccess, args=[StringLit("hello")])

Phase 2: Binder
────────────────────────────────────────────────────────────
BindCallExpression:
  expr = BindExpression(syntax.Expression)  // member access chain
  // expr 当前为 MemberAccessExpression:
  //   Expression = MemberAccessExpression:
  //     Expression = NameExpression("System")
  //     MemberName = "Console"
  //   MemberName = "WriteLine"

  → 解析 MemberAccessExpression 链:
    1. BindNameExpression("System") → 失败（不是变量）
    2. 尝试作为类型名: MetadataResolver.ResolveType("System") → 找到命名空间
    3. 继续解析 ".Console":
       ResolveType("System.Console") → TypeDefinition (static class)
    4. 继续解析 ".WriteLine":
       ResolveStaticMethod(type, "WriteLine", [StringType])
       → 匹配重载 WriteLine(String) → MethodReference

  → 参数绑定:
    BindConversion("hello", StringType) → BoundLiteralExpression

  → 返回 BoundCallExpression:
    Function = MethodReference(System.Console::WriteLine(string))
    Arguments = [BoundLiteralExpression("hello")]
    IsInteropCall = true

Phase 3: Emitter
────────────────────────────────────────────────────────────
EmitCallExpression:
  if (node.IsInteropCall):
    foreach (arg in node.Arguments):
      EmitExpression(ilProcessor, arg)   // ldstr "hello"
    ilProcessor.Emit(OpCodes.Call, node.InteropTarget)
    // → call System.Console.WriteLine(string)
```

```csharp
private void EmitTryStatement(ILProcessor ilProcessor, BoundTryStatement node)
{
    var tryStart = ilProcessor.Body.Instructions.Count;
    foreach (var stmt in node.Body.Statements)
        EmitStatement(ilProcessor, stmt);
    var tryEnd = ilProcessor.Body.Instructions.Count;

    // 如果 try 之后有 finally，直接 fall through 到 finally（leave 处理）
    if (node.FinallyBody != null)
    {
        // 离开 try 块需要 leave 指令跳转到 finally 之后
        var afterFinally = Instruction.Create(OpCodes.Nop);
        ilProcessor.Emit(OpCodes.Leave, afterFinally);
    }

    foreach (var catchBlock in node.Catches)
    {
        var handlerStart = ilProcessor.Body.Instructions.Count;

        // 存储异常对象到 catch 变量
        if (catchBlock.Variable != null)
        {
            var varDef = new VariableDefinition(_knownTypes[catchBlock.ExceptionType]);
            ilProcessor.Body.Variables.Add(varDef);
            _locals[catchBlock.Variable] = varDef;
            ilProcessor.Emit(OpCodes.Stloc, varDef);
        }
        else
            ilProcessor.Emit(OpCodes.Pop); // 不捕获变量则弹出异常

        foreach (var stmt in catchBlock.Body.Statements)
            EmitStatement(ilProcessor, stmt);

        var handlerEnd = ilProcessor.Body.Instructions.Count;
        ilProcessor.Emit(OpCodes.Leave, afterFinally ?? Instruction.Create(OpCodes.Nop));

        // 注册异常处理块
        var handler = new ExceptionHandler(ExceptionHandlerType.Catch)
        {
            TryStart = ilProcessor.Body.Instructions[tryStart],
            TryEnd = ilProcessor.Body.Instructions[tryEnd],
            HandlerStart = ilProcessor.Body.Instructions[handlerStart],
            HandlerEnd = ilProcessor.Body.Instructions[handlerEnd],
            CatchType = _knownTypes[catchBlock.ExceptionType],
        };
        ilProcessor.Body.ExceptionHandlers.Add(handler);
    }

    if (node.FinallyBody != null)
    {
        var finallyStart = ilProcessor.Body.Instructions.Count;
        foreach (var stmt in node.FinallyBody.Statements)
            EmitStatement(ilProcessor, stmt);
        ilProcessor.Emit(OpCodes.Endfinally);

        ilProcessor.Body.ExceptionHandlers.Add(new ExceptionHandler(ExceptionHandlerType.Finally)
        {
            TryStart = ilProcessor.Body.Instructions[tryStart],
            TryEnd = ilProcessor.Body.Instructions[tryEnd],
            HandlerStart = ilProcessor.Body.Instructions[finallyStart],
            HandlerEnd = ilProcessor.Body.Instructions.Count,
        });
    }

    if (afterFinally != null)
        ilProcessor.Append(afterFinally);
}
```

```csharp
protected override BoundStatement RewriteUsingStatement(BoundUsingStatement node)
{
    var disposeCall = new BoundCallExpression(node.Syntax,
        MemberAccessExpression(node.Syntax, Variable(node.Syntax, node.Variable),
            "Dispose", TypeSymbol.Void),
        ImmutableArray<BoundExpression>.Empty);

    var nullCheck = new BoundBinaryExpression(node.Syntax,
        Variable(node.Syntax, node.Variable),
        BoundBinaryOperator.Bind(SyntaxKind.BangEqualsToken, node.Variable.Type, TypeSymbol.Any)!,
        new BoundLiteralExpression(node.Syntax, null));

        new BoundExpressionStatement(node.Syntax, disposeCall), null);

    var finallyBody = new BoundBlockStatement(node.Syntax,
        ImmutableArray.Create<BoundStatement>(disposeIfNotNull));

    var tryStmt = new BoundTryStatement(node.Syntax, node.Body,
        ImmutableArray<BoundCatchBlock>.Empty, finallyBody);

    return RewriteStatement(new BoundBlockStatement(node.Syntax,
        ImmutableArray.Create<BoundStatement>(node.Declaration, tryStmt)));
```

```csharp
private MemberSyntax ParseMember()
{
    return Current.Kind switch
    {
        SyntaxKind.FunctionKeyword => ParseFunctionDeclaration(),
        SyntaxKind.EnumKeyword => ParseEnumDeclaration(),
        SyntaxKind.ClassKeyword => ParseClassDeclaration(),
        SyntaxKind.InterfaceKeyword => ParseInterfaceDeclaration(),

private MemberSyntax ParsePossiblyAttributedMember()
{
    var modifiers = ParseModifiers(); // 读取多个修饰符

    if (Current.Kind == SyntaxKind.InterfaceKeyword)

    if (Current.Kind == SyntaxKind.FunctionKeyword)
        return ParseFunctionDeclaration(modifiers);
    if (Current.Kind == SyntaxKind.EnumKeyword)

    // treat as global statement with modifiers (error if invalid)
```

```csharp
private void BindClassDeclaration(ClassDeclarationSyntax syntax)
{
    var scope = new BoundScope(_scope); // 类作用域

    // 1. 注册类型自身（支持递归）
    var classType = new ClassType(syntax.Identifier.Text, syntax, null,
        ImmutableArray<FunctionSymbol>.Empty, ImmutableArray<VariableSymbol>.Empty,
        ImmutableArray<FunctionSymbol>.Empty);
    _scope.TryDeclareType(classType);

    // 2. 解析基类
    TypeSymbol? baseType = null;
    if (syntax.BaseType != null)
    {
        baseType = BindTypeClause(syntax.BaseType);
        classType.BaseType = baseType;
    }

    // 3. 绑定成员（字段、属性、方法）
    var fieldsBuilder = ImmutableArray.CreateBuilder<VariableSymbol>();
    var methodsBuilder = ImmutableArray.CreateBuilder<FunctionSymbol>();

    foreach (var member in syntax.Members)
    {
        switch (member.Kind)
        {
            case SyntaxKind.FieldDeclaration:
                var fieldDecl = (FieldDeclarationSyntax)member;
                var field = BindFieldDeclaration(fieldDecl, classType);
                fieldsBuilder.Add(field);
                scope.TryDeclareVariable(field);
                break;
            case SyntaxKind.FunctionDeclaration:
                var funcDecl = (FunctionDeclarationSyntax)member;
                // 标记为成员方法（隐含第一个参数 this）
                BindMethodDeclaration(funcDecl, classType);
                break;
            // ...
        }
    }

    classType.Fields = fieldsBuilder.ToImmutable();
    classType.Methods = methodsBuilder.ToImmutable();
}
```

```csharp
    _assemblyDefinition.MainModule.Types.Add(typeDef);

    // Emit 字段
    foreach (var field in classType.Fields)
    {
        var fieldDef = new FieldDefinition(field.Name,
            FieldAttributes.Private, GetTypeReference(field.Type));
        typeDef.Fields.Add(fieldDef);
    }

    // Emit 方法（含 .ctor）
    foreach (var method in classType.Methods)
    {
        var methodDef = EmitClassMethod(classType, method);
        typeDef.Methods.Add(methodDef);
    }

    return typeDef;
}

// 实例方法调用 emit：需要将 this 作为第一个参数（ldarg 0）
private void EmitInstanceMethodCall(ILProcessor ilProcessor, BoundCallExpression node)
{
    // ldarg 0 = this
    ilProcessor.Emit(OpCodes.Ldarg_0);

    foreach (var arg in node.Arguments)
        EmitExpression(ilProcessor, arg);

    if (node.IsVirtualCall)
        ilProcessor.Emit(OpCodes.Callvirt, node.InteropTarget);
    else
        ilProcessor.Emit(OpCodes.Call, node.InteropTarget);
}
```

### 8.3 继承与多态

**继承链解析**（Binder）：
```csharp
// 验证继承合法性
private void ValidateInheritance(ClassDeclarationSyntax syntax, ClassType classType)
{
    if (classType.BaseType == null) return;

    // 1. 基类不能是 sealed
    if (classType.BaseType is ClassType baseClass && baseClass.IsSealed)
    {
        _diagnostics.ReportCannotDeriveFromSealedClass(
            syntax.BaseType!.Location, baseClass.Name);
        return;
    }

    // 2. 继承深度限制（防循环）
    var depth = 1;
    var current = classType.BaseType;
    while (current is ClassType ct)
    {
        if (ct == classType)
        {
            _diagnostics.ReportCircularBaseClass(syntax.Identifier.Location);
            return;
        }
        current = ct.BaseType;
        if (++depth > 100)
        {
            _diagnostics.ReportInheritanceDepthExceeded(syntax.Identifier.Location);
            return;
        }
    }

    // 3. 抽象类规则：如果基类是抽象类，派生类必须实现所有抽象成员
    if (baseClass != null && baseClass.IsAbstract)
    {
        foreach (var abstractMethod in baseClass.Methods
            .Where(m => m.IsAbstract))
        {
            var overrideMethod = classType.Methods
                .FirstOrDefault(m => m.Name == abstractMethod.Name &&
                    m.Parameters.Select(p => p.Type)
                        .SequenceEqual(abstractMethod.Parameters.Select(p => p.Type)));
            if (overrideMethod == null || !overrideMethod.IsOverride)
            {
                _diagnostics.ReportMissingAbstractImplementation(
                    syntax.Identifier.Location, baseClass.Name, abstractMethod.Name);
            }
        }
    }
}
```

**虚方法表（VTable）设计**：
```csharp
internal sealed class VirtualMethodTable
{
    private readonly Dictionary<string, List<FunctionSymbol>> _methods = new();

    public void AddMethod(FunctionSymbol method)
    {
        var key = BuildKey(method.Name, method.Parameters);
        if (!_methods.ContainsKey(key))
            _methods[key] = new List<FunctionSymbol>();
        _methods[key].Add(method);
    }

    public FunctionSymbol? Resolve(string name, ImmutableArray<TypeSymbol> paramTypes, ClassType instanceType)
    {
        // 从实例类型开始向上搜索，最匹配的 override 优先
        var current = instanceType;
        while (current != null)
        {
            var key = BuildKey(name, paramTypes);
            if (_methods.TryGetValue(key, out var methods))
            {
                var match = methods.FirstOrDefault(m =>
                    m.DeclaringType == current && m.IsOverride == false);
                if (match != null) return match;
            }
            current = current.BaseType as ClassType;
        }
        return null;
    }

    private static string BuildKey(string name, ImmutableArray<ParameterSymbol> parameters) =>
        $"{name}({string.Join(",", parameters.Select(p => p.Type.Name))})";
}
```

**Emitter — 虚方法**：
```csharp
// 虚方法表在 IL 中通过 callvirt 实现
// 非虚方法用 call，虚方法用 callvirt
private void EmitCallExpression(ILProcessor ilProcessor, BoundCallExpression node)
{
    if (node.IsInstanceMethod)
    {
        // 加载 this
        EmitExpression(ilProcessor, node.Instance);

        foreach (var arg in node.Arguments)
            EmitExpression(ilProcessor, arg);

        if (node.IsVirtualCall)
            ilProcessor.Emit(OpCodes.Callvirt, GetMethodReference(node.Function));
        else
            ilProcessor.Emit(OpCodes.Call, GetMethodReference(node.Function));
    }
    else
    {
        // static method (existing logic)
        foreach (var arg in node.Arguments)
            EmitExpression(ilProcessor, arg);
        ilProcessor.Emit(OpCodes.Call, GetMethodReference(node.Function));
    }
}
```

```csharp
        MethodAttributes.SpecialName | MethodAttributes.RTSpecialName,
        _knownTypes[TypeSymbol.Void]);
```

```csharp
internal sealed class ImportTable
{
    private readonly ImmutableArray<string> _imports; // ["System", "System.Collections.Generic"]

    public ImportTable(ImmutableArray<string> imports)
    {
        _imports = imports;
    }

    /// <summary>尝试用导入前缀补齐为完全限定名</summary>
    public TypeSymbol? ResolveType(string name, MetadataResolver resolver)
    {
        // 1. 先尝试直接作为全名
        var fullType = resolver.ResolveType(name);
        if (fullType != null) return fullType;

        // 2. 逐个尝试 import 前缀
        foreach (var import in _imports)
        {
            var qualified = $"{import}.{name}";
            fullType = resolver.ResolveType(qualified);
            if (fullType != null) return fullType;
        }

        return null;
    }
}
```

```csharp
private MemberSyntax ParseUsingDirective()
{
    var keyword = MatchToken(SyntaxKind.UsingKeyword);
    var name = ParseQualifiedName(); // "System" 或 "System.Collections.Generic"
    return new UsingDirectiveSyntax(_syntaxTree, keyword, name);
}

private MemberSyntax ParseNamespaceDeclaration()
{
    var keyword = MatchToken(SyntaxKind.NamespaceKeyword);
    var name = ParseQualifiedName(); // "MyApp.Models"
    var openBrace = MatchToken(SyntaxKind.OpenBraceToken);
    var members = ImmutableArray.CreateBuilder<MemberSyntax>();

    while (Current.Kind != SyntaxKind.CloseBraceToken &&
           Current.Kind != SyntaxKind.EndOfFileToken)
    {
        members.Add(ParseMember());
    }

    var closeBrace = MatchToken(SyntaxKind.CloseBraceToken);
    return new NamespaceDeclarationSyntax(_syntaxTree, keyword, name,
        openBrace, members.ToImmutable(), closeBrace);
}

/// <summary>解析点分限定名称：A.B.C</summary>
private NameSyntax ParseQualifiedName()
{
    var identifier = MatchToken(SyntaxKind.IdentifierToken);

    if (Current.Kind == SyntaxKind.DotToken)
    {
        var dot = NextToken();
        var right = ParseQualifiedName(); // 递归解析剩余部分
        return new QualifiedNameSyntax(_syntaxTree, identifier, dot, right);
    }

    return new SimpleNameSyntax(_syntaxTree, identifier);
}
```

**Emitter**：命名空间声明不影响 IL 生成（IL 中的类型总是完全限定名）。`using` 不影响 emit，仅在 Binder 中影响名称解析。

**依赖关系**：依赖 Phase 2 成员访问 `.` 和 Phase 3 MetadataResolver（类型全名解析）。

**注意事项**：
- Cocoa 不支持 `using static`、`using alias`（初始版本）
- 命名空间不影响运行时行为，只在编译期影响名称解析
- 函数声明在命名空间内时，生成的 IL 类使用 `Namespace.ClassName` 全名
