# Cocoa 开发计划 — 版本路线图

> 基准：C# 13 (.NET 9.0 / CLR)  
> 当前版本：**v0.1**（Evaluator 解释执行，基础类型/语句/函数）
> 文档：`docs/语法手册.md` — 语言参考 · `docs/实现目标.md` — 详细设计

## v0.x — 预发布：对齐 C# 1.0 语法 + 类型系统 + .NET 互操作

### v0.1 ✓ (当前状态)

| 类别 | 功能 |
|------|------|
| **类型** | `int` `bool` `string` `double` `void` `any` |
| **变量** | `let` 不可变 / `var` 可变，类型推断 |
| **运算符** | 算术 `+ - * /`、比较 `== != < > <= >=`、逻辑 `! && \|\|`、位 `~ & \| ^`、赋值 `= += -= *= /= &= \|= ^=` |
| **语句** | `if/else` `while` `do-while` `for i = a to b` `break` `continue` `return` `{ }` |

| **内置** | `print` `input` `random` |
| **注释** | `//` `/* */` |
| **执行** | Evaluator (解释执行) |

### v0.2.5 — 工具链：项目文件 (.coproj/.cosln) + 编译手册

```
coc MyApp.coproj              // 编译项目文件
coc MyApp.cosln                // 编译解决方案
coc MyApp.coproj --release     // 发布模式编译
```

| 新增 | 详情 |
|------|------|
| **`.coproj` 项目文件** | JSON 格式，18 字段：`name` `outputType` `sources` `references` `projectReferences` `packageReferences` `outputPath` `moduleName` `entryPoint` `debug` `define` `deterministic` `treatWarningsAsErrors` `version` `fileVersion` `company` `copyright` `description` |
| **`.cosln` 解决方案文件** | JSON 格式，支持 `name` `projects` `folders`（逻辑分组）`configurations`（自定义配置）`projectConfig`（按项目/配置覆盖 build 标志），编译顺序由 projectReferences 依赖图拓扑排序自动推断 |
| **projectReferences** | 跨项目引用：A → B.coproj，自动解析 B 的输出 DLL 路径作为 A 的 MetadataReference，支持依赖图拓扑排序和循环依赖检测 |
| **packageReferences** | NuGet 包引用：`{"Newtonsoft.Json": "13.0.3"}`，搜索本地缓存 + 全局用户缓存 + NuGet assets 文件 |
| **Debug/Release 模式** | `--debug`（默认）：生成 PDB + 禁用 JIT 优化；`--release`：无 PDB + 启用 JIT 优化。项目文件 `debug` 字段可配，命令行可覆盖 |
| **程序集元数据** | `version` `fileVersion` `company` `copyright` `description` 字段控制 CLR 程序集属性 |
| **条件编译** | `define` 数组字段，配合 `#if`/`#else`/`#endif` 预处理指令 |
| **ProjectLoader** | `src/Cocoa/CodeAnalysis/ProjectSystem/ProjectLoader.cs` — 解析 JSON、展开 glob、解析三种引用（DLL/coproj/NuGet）、拓扑排序、计算输出路径、合并命令行覆盖 |
| **`编译手册.md`** | 编译器使用文档，命令行参数说明、项目/解决方案格式详解（含全部新增字段）、快速入门示例、构建流程说明 |
| **编译流程** | `.co` → 直接编译；`.coproj` → LoadProject + LoadSyntaxTrees + LoadMetadataReferences（含 projectRef/packageRef 解析）+ Compilation.Create + Emit；`.cosln` → LoadSolution → 配置过滤 → 拓扑排序 → 按序编译 |
